# LOLWeb

<a href="https://github.com/L0Lweb/WebShellBins"><img align="right" src="assets/logo.png" style="width: 100px" /></a>

**LOLWeb** is a curated list of web shells and payloads for various web servers and languages (Jenkins, Tomcat, WordPress, GitLab, etc.).

It is based on the [LOLWeb](https://github.com/L0Lweb/WebShellBins) structure but focused on **web payloads**.

## Features

- **Copy-Paste Ready**: Quickly find and copy payloads.
- **LHOST/LPORT Templating**: Enter your attacker IP/Port once, and all payloads update automatically.
- **Clean Interface**: Uses the familiar LOLWeb layout.
